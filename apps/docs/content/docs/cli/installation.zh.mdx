---
title: CLI Installation
description: Install the Multica CLI and start the local daemon.
---

## Homebrew

```bash
brew install multica-ai/tap/multica
```

## Build from Source

```bash
git clone https://github.com/multica-ai/multica.git
cd multica
make build
cp server/bin/multica /usr/local/bin/multica
```

## Configure

```bash
multica setup self-host
```

For custom ports:

```bash
multica setup self-host --server-url http://localhost:8080 --app-url http://localhost:3000
```

This writes local CLI configuration, discovers workspaces, and starts the daemon. There is no browser login or token exchange.

## Verify

```bash
multica daemon status
multica workspace list
```

If no runtimes are detected, install at least one supported agent CLI on PATH and restart the daemon.
