## Prerequisites

- Node.js 22
- pnpm 10.28+
- Go 1.26.1
- Podman or Docker Compose

## Quick Start

```bash
make dev
```

This sets up the environment, starts PostgreSQL, runs migrations, and starts the backend and frontend.

## Worktrees

```bash
git worktree add ../multica-feature -b feat/my-change main
cd ../multica-feature
make worktree-env
make dev
```

Worktrees use `.env.worktree`, unique databases, and unique ports. Do not copy `.env` into a worktree.

## Commands

```bash
pnpm typecheck
pnpm test
pnpm lint
pnpm build

make test
make check
make sqlc
make migrate-up
make migrate-down
make build
```

## Compose

The default compose command is `podman compose`.

```bash
COMPOSE="docker compose" make dev
```

## Local Daemon

```bash
make build
./server/bin/multica setup self-host
./server/bin/multica daemon start --foreground
```

No login or token setup is required in solo local mode.
